Embedded Boundaries¶

IAMR uses AMReX’s Embedded Boundary (cut cell) functionality to represent internal or external geometry in the flow. See the AMReX EB documentation (Overview of Embedded Boundary Description) for more details on AMReX’s approach to embedded boundaries.
